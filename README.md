# CPU & GPU Simulator

An interactive CPU and GPU simulator for learning assembly language and understanding how processors work.

## Live Demo

**https://tyler-french.github.io/cpu-sim/**

[Learn more about this project](https://tyler-french.github.io/cpu-sim/about.html)

## Features

- Visual CPU pipeline (fetch/decode/execute/writeback)
- 8 general-purpose registers and 256 bytes of memory
- GPU with 8 parallel processing cores and 128 bytes of VRAM
- Step-by-step or continuous execution with speed control
- Built-in example programs

## Run Locally

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```
